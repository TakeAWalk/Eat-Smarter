Eat Smarter

Provices APIs (REST) to retrieve/update information to and from Eat Smarter database

GET Requests

Testing workflow, hit the following endpoints in order: /api/createUser, /api/associateAllegry, /api/getUser

  • GET - api/getMealPlan - Gets a one day meal plan which includes breakfast, lunch and dinner meals.
  • GET - api/createUser - Creates a static user
  • GET - api/associateAllegry - Creates a static allegry and associdates the allegry to the user created in the previous endpoint.
  • GET - api/getUser - Gets all users and associated allergies.
